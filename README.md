WebCraft – Dynamic Website Builder

This application is a platform that allows users to create their own personal or business websites without writing any code.
Features
User Authentication System: Sign up, log in, and manage your websites.
Drag-and-Drop Interface: Section-based website editor.
Dynamic Content: All texts and colors can be updated instantly.
Templates: Hero, About Us, Services, and Contact sections.
Modern Design: Glassmorphism and premium aesthetics.

Installation

Database Setup:
Create a database named web_builder_db via MySQL (PhpMyAdmin, etc.).
Run the content of the db_setup.sql file in the SQL tab.

Connection Settings:
Open the config/db.php file.
Check your database username and password (root and empty by default).

Run the Application:
Start Apache and MySQL on XAMPP.
Navigate to localhost/Web Sites Tasarlayan Web Application/ in your browser.
Usage

First, create an account.
From the dashboard, click the "Create New Site" button.
Give your site a name and a unique URL (slug).
In the editor screen, click on sections to edit their content and colors.
Expand your site by adding new sections.


WebCraft - Dinamik Web Sitesi Oluşturucu

Bu uygulama, kullanıcıların kendi kişisel veya işletme web sitelerini kod yazmadan oluşturmalarına olanak tanıyan bir platformdur.

## Özellikler
- **Üyelik Sistemi:** Kayıt ol, giriş yap ve sitelerini yönet.
- **Sürükle-Bırak Mantığı:** Bölüm bazlı site düzenleyici.
- **Dinamik İçerik:** Tüm metinler ve renkler anında güncellenebilir.
- **Şablonlar:** Hero, Hakkımızda, Hizmetler ve İletişim bölümleri.
- **Modern Tasarım:** Glassmorphism ve premium estetik.

## Kurulum
1. **Veritabanı Oluşturma:**
   - MySQL (PhpMyAdmin vb.) üzerinden `web_builder_db` adında bir veritabanı oluşturun.
   - `db_setup.sql` dosyasındaki içeriği SQL sekmesinde çalıştırın.
2. **Bağlantı Ayarları:**
   - `config/db.php` dosyasını açın.
   - Veritabanı kullanıcı adınızı ve şifrenizi (`root` ve ` ` varsayılan) kontrol edin.
3. **Çalıştırma:**
   - XAMPP üzerinde Apache ve MySQL'i başlatın.
   - Tarayıcıdan `localhost/Web Sites Tasarlayan Web Application/` adresine gidin.

## Kullanım
- Önce bir hesap oluşturun.
- Dashboard üzerinden "Yeni Site Oluştur" butonuna basın.
- Sitenize bir isim ve benzersiz bir URL (slug) verin.
- Düzenleyici ekranında bölümlerin üzerine tıklayarak içeriklerini ve renklerini değiştirin.
- Yeni bölümler ekleyerek sitenizi genişletin.
